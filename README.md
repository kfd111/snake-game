# 🐍 Snake Game

A classic Snake game built with **HTML5**, **CSS3**, and **JavaScript**.

## Features

✨ **Game Features:**
- Classic snake gameplay with smooth controls
- Colorful graphics with visual effects
- Score tracking with high score saved locally
- Difficulty increases as you progress
- Pause and resume functionality
- Responsive design for mobile and desktop

## How to Play

1. Click **"Start Game"** to begin
2. Use **Arrow Keys** or **WASD** to control the snake
3. Eat the **red food** to grow longer and earn points
4. **Don't hit the walls** or your own body!
5. Try to achieve the **highest score**

## Controls

- **↑ / W** - Move Up
- **↓ / S** - Move Down
- **← / A** - Move Left
- **→ / D** - Move Right
- **Start** - Begin the game
- **Pause** - Pause/Resume the game
- **Reset** - Reset the game to initial state

## Game Rules

- Each food eaten = +10 points
- Snake grows by 1 segment when food is eaten
- Game speed increases every 50 points
- Game ends if snake hits walls or itself
- High score is automatically saved
